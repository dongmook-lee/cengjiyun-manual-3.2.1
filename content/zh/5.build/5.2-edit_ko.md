빌드 작업을 편집한다.

### a\) 좌측 메인 메뉴 → 빌드 → 빌드 액션 메뉴\(활성화\) → [작업 설정 편집]을 선택하여 해당 페이지로 이동한다.
### b\) 기본 정보, 어플리케이션 다운로드, 어플리케이션 빌드, 이미지 빌드 내용을 수정하고 [수정] 버튼을 눌러 작업 설정을 편집한다.
