---
login는 설치된 Cluster의 Cocktail DashBoard 또는 Cocktail Online에 로그인 한다.

---
### Prerequisite
Cube Cluster 설치 상태.`create` command 


### Usage

* `cube login [flags]`

### Flags

##### url

* `--url, -u`

* 접속할 Cocktail DashBoard의 Url을 입력 한다.
* 필수: 필수

##### system

* `--system, -s`

* System 정보를 입력 한다.

* 필수: 필수


##### id

* `--id, -i`

* ID 정보를 입력 한다.

* 필수: 필수


##### password

* `--password, -p`

* password 정보를 입력 한다.

* 필수: 필수
